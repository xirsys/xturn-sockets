# XTurn Sockets

Socket system providing flexible UDP, TCP, TLS and DTLS functionality.

Installation
===
This package can be installed by adding `xturn_sockets` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [{:xturn_sockets, "~> 1.0.0"}]
end
```
